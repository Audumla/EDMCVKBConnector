# Security Policy - Agent System

## Supported Versions

Currently, the following versions of the Agent System are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it by opening a GitHub Issue.

We aim to acknowledge reports within 48 hours and provide a timeline for fixes where appropriate.
