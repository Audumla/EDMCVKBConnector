- logs/transcripts

## Smoke Test

After any major change run a quick smoke test to verify the project still compiles and has no syntax errors.
Fix all failures before recording the changelog or marking the task complete.

**Python** — syntax-check each changed file:
```
python -m py_compile path/to/changed_file.py
```
Or verify imports: `python -c "import <module_name>"`

**TypeScript / JavaScript**:
```
npx tsc --noEmit
node --check file.js
```

**Shell scripts**: `bash -n script.sh`

**General**: `<entry-point> --help` (or `--version`)

If a test suite exists, also run it: `python -m pytest tests/ -q` or the project-equivalent.
